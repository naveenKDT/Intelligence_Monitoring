# Frontend - Intelligence Monitoring Platform

React + TypeScript + Material UI frontend for the Intelligence Monitoring Platform.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=/api/v1
```

## Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Tech Stack

- React 18
- TypeScript
- Material UI (MUI)
- AG Grid Community
- ECharts
- TanStack Query
- React Router v6
- Vite
